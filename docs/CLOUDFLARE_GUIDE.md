# Настройка Cloudflare для CDN-режима

CDN-режим проксирует VPN-трафик через Cloudflare, обходя IP-блокировки. Клиент подключается к IP Cloudflare (не блокируется), а Cloudflare пересылает трафик на ваш VPS.

## 1. Купите домен

Подойдет любой дешевый домен. Варианты:

- **Cloudflare Registrar** — без наценки, сразу в Cloudflare
- **Namecheap** — домены `.xyz` / `.site` от $1-2/год

## 2. Добавьте домен в Cloudflare

1. Зарегистрируйтесь на [cloudflare.com](https://cloudflare.com) (бесплатный план)
2. Нажмите **Add a site** и введите ваш домен
3. Выберите план **Free**
4. Cloudflare покажет NS-серверы — пропишите их у вашего регистратора домена
5. Дождитесь активации (обычно 5-30 минут)

## 3. Создайте A-запись

1. Перейдите в **DNS** → **Records**
2. Нажмите **Add record**:
   - **Type**: `A`
   - **Name**: `@` (или поддомен, например `vpn`)
   - **IPv4 address**: IP вашего VPS
   - **Proxy status**: **Proxied** (оранжевое облако включено)
3. Нажмите **Save**

> Если вы указали поддомен `vpn`, ваш домен будет `vpn.example.com` — используйте его в скрипте.

## 4. Настройте SSL/TLS

1. Перейдите в **SSL/TLS** → **Overview**
2. Выберите режим **Flexible**

> **Почему Flexible?** XRay слушает на порту 2082 без TLS. Cloudflare сам терминирует TLS от клиента, а до вашего сервера ходит по HTTP. VLESS-payload и так зашифрован, двойное шифрование не нужно.

## 5. Проверка

1. Подключитесь через NfrCDN-ссылку в Streisand / V2Box
2. Откройте `ifconfig.me` — должен показать IP вашего VPS (не Cloudflare)
3. Если не работает:
   - Проверьте что домен резолвится: `nslookup ваш-домен` — должен показать IP Cloudflare (не VPS)
   - Проверьте что порт 2082 открыт на сервере: `ufw status`
   - Проверьте XRay: `journalctl -u xray -n 20`
   - Убедитесь что SSL/TLS стоит на **Flexible**, а не Full
