# Настройка Cloudflare для CDN-режима

CDN-режим проксирует VPN-трафик через Cloudflare, обходя IP-блокировки. Клиент подключается к IP Cloudflare на порту 443 (не блокируется), а Cloudflare пересылает трафик на ваш VPS.

**Схема работы:** клиент → Cloudflare:443 → VPS:443 (nginx SNI-роутер) → XRay:2053 (gRPC+TLS)

## 1. Купите домен

Подойдёт любой дешёвый домен:

- **Cloudflare Registrar** — без наценки, сразу в Cloudflare
- **Namecheap** — домены `.xyz` / `.site` от $1-2/год

## 2. Добавьте домен в Cloudflare

1. Зарегистрируйтесь на [cloudflare.com](https://cloudflare.com) (бесплатный план)
2. Нажмите **Add a site** и введите ваш домен
3. Выберите план **Free**
4. Cloudflare покажет NS-серверы — пропишите их у вашего регистратора домена
5. Дождитесь активации (обычно 5-30 минут)

## 3. Создайте A-запись

1. Перейдите в **DNS** → **Records**
2. Нажмите **Add record**:
   - **Type**: `A`
   - **Name**: поддомен, например `cdn` (итоговый адрес: `cdn.example.com`)
   - **IPv4 address**: IP вашего VPS
   - **Proxy status**: **Proxied** (оранжевое облако включено)
3. Нажмите **Save**

> Используйте точно тот домен/поддомен, который вводили в скрипте установки.

## 4. Настройте SSL/TLS

1. Перейдите в **SSL/TLS** → **Overview**
2. Выберите режим **Full**

> **Почему Full?** XRay слушает на порту 2053 с TLS (самоподписанный сертификат). Cloudflare должен подключаться к серверу по HTTPS, иначе соединение не установится. Режим Flexible здесь не подойдёт.

## 5. Включите gRPC

1. Перейдите в **Network**
2. Включите переключатель **gRPC**

> Это необходимо для работы gRPC-протокола через Cloudflare.

## 6. Проверка

1. Подключитесь через CDN-ссылку в Streisand / V2Box
2. Откройте `ifconfig.me` — должен показать IP вашего VPS (не Cloudflare)
3. Если не работает:
   - Проверьте что XRay и nginx слушают нужные порты: `ss -tlnp | grep -E '2053|9443|:443'`
   - Проверьте конфиг nginx: `nginx -t`
   - Проверьте статус nginx: `systemctl status nginx`
   - Проверьте статус XRay: `journalctl -u xray -n 20`
   - Убедитесь что SSL/TLS стоит на **Full**, а не Flexible
   - Убедитесь что gRPC включён в **Network**
   - Убедитесь что домен резолвится через Cloudflare: `nslookup ваш-домен` — должен показать IP Cloudflare (не VPS)
