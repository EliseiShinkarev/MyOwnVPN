# Настройка клиентов

После запуска `setup.sh` вы получите одну или две ссылки:

**Reality (основная):**
```
vless://UUID@IP:443?encryption=none&flow=xtls-rprx-vision&security=reality&sni=www.microsoft.com&fp=chrome&pbk=...&sid=...&type=tcp#MyVPN
```

**CDN (если включён CDN-режим):**
```
vless://UUID@домен:443?encryption=none&security=tls&sni=домен&type=ws&host=домен&path=%2F...&fp=chrome#MyVPN-CDN
```

## iPhone

### Streisand (рекомендуется)

1. Установите [Streisand](https://apps.apple.com/app/streisand/id6450534064) из App Store (бесплатно)
2. Откройте приложение
3. Нажмите **+** в правом верхнем углу
4. Выберите **Добавить из буфера обмена** (если скопировали ссылку) или **Сканировать QR-код** (если видите QR на экране)
5. Появится сервер «MyVPN» — нажмите на него для подключения
6. Разрешите добавление VPN-конфигурации при первом подключении

### Проверка

- Откройте Safari → любой сайт
- Проверьте IP: зайдите на `ifconfig.me` — должен показать IP вашего VPS

## Mac

### V2Box

1. Установите [V2Box](https://apps.apple.com/app/v2box-v2ray-client/id6446814690) из App Store
2. Откройте приложение
3. Нажмите **+** → **Import from clipboard**
4. Скопируйте `vless://` ссылку и вставьте
5. Выберите сервер и нажмите **Connect**

### Streisand для Mac

1. Установите [Streisand](https://apps.apple.com/app/streisand/id6450534064) из App Store
2. Нажмите **+** → **Добавить из буфера обмена**
3. Подключитесь к серверу

## Какую ссылку использовать

| Ситуация | Ссылка |
|----------|--------|
| Wi-Fi, домашний интернет, большинство случаев | **MyVPN** (Reality) — быстрее |
| Мобильный интернет в центре Москвы, IP-блокировки | **MyVPN-CDN** — обходит блокировку IP |
| Не уверены | Попробуйте Reality, если не работает — CDN |

> CDN-ссылка немного медленнее из-за дополнительного хопа через Cloudflare, но работает даже при IP-блокировках.

## Советы

- **Включайте VPN перед использованием** — он не расходует батарею, когда нет трафика
- **Если перестало работать** — попробуйте переподключиться (выкл/вкл)
- **Скорость зависит от VPS** — для видео в 4K нужен сервер с каналом от 200 Мбит/с
- **Добавьте обе ссылки** — если включён CDN-режим, добавьте обе ссылки в клиент и переключайтесь при необходимости
