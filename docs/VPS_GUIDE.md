# Где арендовать VPS

## Требования

- **ОС**: Ubuntu 22.04 или 24.04
- **Ресурсы**: 1 vCPU, 512 MB RAM, 10 GB SSD (минимум)
- **Локация**: Европа (Финляндия, Нидерланды, Германия — пинг из Москвы 20-40 мс)

## Провайдеры, принимающие оплату из РФ

### Aeza.net
- Серверы: Нидерланды, Финляндия, Германия, Швеция
- Оплата: российские карты, СБП, криптовалюта
- Цена: от ~3-5 €/мес за минимальный VPS
- Удобная панель, быстрое создание сервера

### Timeweb Cloud
- Серверы: Нидерланды, Польша, Казахстан
- Оплата: российские карты, СБП
- Цена: от ~300 ₽/мес
- Российская компания, русскоязычная поддержка

### 4VPS.su
- Серверы: Нидерланды, Германия
- Оплата: российские карты, ЮMoney, криптовалюта
- Цена: от ~2-3 €/мес
- Бюджетный вариант

## Рекомендации по выбору локации

| Страна | Пинг из Москвы | Комментарий |
|--------|---------------|-------------|
| Финляндия | ~20 мс | Лучший пинг, ближайшая |
| Нидерланды | ~30-40 мс | Хороший баланс цена/качество |
| Германия | ~30-40 мс | Много провайдеров |

## Заметка для московских операторов

Мобильные операторы в центре Москвы могут блокировать VPN по IP (белые списки). Если Reality-подключение не работает через мобильный интернет, используйте **CDN-режим** — трафик пойдёт через Cloudflare, чьи IP не блокируются. Для этого нужен домен (от $1-2/год) и бесплатный аккаунт Cloudflare. Подробнее — в [CLOUDFLARE_GUIDE.md](CLOUDFLARE_GUIDE.md).

## После покупки

1. Выберите Ubuntu 22.04 или 24.04 при создании сервера
2. Получите IP-адрес и root-пароль (обычно на почту)
3. Подключитесь: `ssh root@ВАШ_IP`
4. Следуйте инструкции в [README.md](../README.md)
